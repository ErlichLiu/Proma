# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via GitHub Security Advisories or email the maintainers directly.

Please include:
- Clear description
- Steps to reproduce
- Potential impact  
- Suggested fix (if available)

## Response Timeline

- Acknowledgment: within 48 hours
- Updates: regular progress reports
