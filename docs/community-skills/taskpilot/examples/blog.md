# 示例项目：技术博客系统

> 使用 TaskPilot Skill 拆解的完整示例

## 项目需求

创建一个现代化的技术博客平台，支持：
- Markdown 编辑和实时预览
- 代码语法高亮
- 用户认证和评论系统
- 响应式设计

## 技术栈

- **前端**: Next.js 14 + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: PostgreSQL + Prisma ORM
- **部署**: Vercel

## TaskPilot 拆解结果

### 阶段 1: 基础架构 (3 个任务)

#### 任务 #1: 初始化 Next.js 项目
**描述**:
- 使用 `create-next-app` 创建项目
- 配置 TypeScript 和 ESLint
- 安装 Tailwind CSS
- 配置 `next.config.js`

**完成标准**: 能够运行 `npm run dev` 并访问首页

**推荐工具**: Cursor (项目初始化)

**预计时间**: 30 分钟

---

#### 任务 #2: 设计数据库 Schema
**描述**:
- 设计用户表 (users)
- 设计文章表 (posts)
- 设计评论表 (comments)
- 创建 Prisma Schema
- 生成迁移文件

**完成标准**: 完成 `schema.prisma` 文件并成功迁移

**推荐工具**: Claude (数据库设计)

**预计时间**: 1 小时

**依赖**: 任务 #1

---

#### 任务 #3: 配置 Tailwind 和 UI 组件库
**描述**:
- 配置 Tailwind CSS 主题
- 安装 ShadcnUI 组件
- 创建基础布局组件 (Header/Footer)
- 配置暗色模式

**完成标准**: 完成基础 UI 框架

**推荐工具**: Cursor (UI 开发)

**预计时间**: 1.5 小时

**依赖**: 任务 #1

---

### 阶段 2: 核心功能 (5 个任务)

#### 任务 #4: 实现 Markdown 编辑器
**描述**:
- 集成 `react-markdown` 或 `MDEditor`
- 实现实时预览功能
- 添加工具栏 (加粗/斜体/链接等)
- 支持图片上传

**完成标准**: 能够编辑和预览 Markdown 文章

**推荐工具**: Cursor (React 组件开发)

**预计时间**: 2 小时

**依赖**: 任务 #3

---

#### 任务 #5: 添加代码高亮功能
**描述**:
- 集成 `highlight.js` 或 `prism.js`
- 支持多种编程语言
- 添加代码复制按钮
- 配置主题样式

**完成标准**: Markdown 中的代码块能够正确高亮

**推荐工具**: Cursor (前端开发)

**预计时间**: 1 小时

**依赖**: 任务 #4

---

#### 任务 #6: 开发文章 CRUD API
**描述**:
- 创建 `/api/posts` 路由
- 实现创建/读取/更新/删除接口
- 添加分页和搜索功能
- 集成 Prisma ORM

**完成标准**: 所有 API 端点能够正常工作

**推荐工具**: Claude Code (API 开发)

**预计时间**: 2.5 小时

**依赖**: 任务 #2

---

#### 任务 #7: 实现用户认证系统
**描述**:
- 集成 NextAuth.js
- 实现邮箱/密码登录
- 添加 GitHub OAuth
- 创建用户个人页面

**完成标准**: 用户能够注册、登录和管理个人信息

**推荐工具**: Claude Code (认证系统)

**预计时间**: 3 小时

**依赖**: 任务 #2

---

#### 任务 #8: 开发评论系统
**描述**:
- 创建评论 API (`/api/comments`)
- 实现评论组件 (嵌套回复)
- 添加点赞功能
- 实现评论通知

**完成标准**: 用户能够发表和回复评论

**推荐工具**: Cursor (全栈开发)

**预计时间**: 2.5 小时

**依赖**: 任务 #6, 任务 #7

---

### 阶段 3: 优化与部署 (4 个任务)

#### 任务 #9: 编写单元测试
**描述**:
- 配置 Jest 和 React Testing Library
- 为 API 路由编写测试
- 为关键组件编写测试
- 达到 80% 代码覆盖率

**完成标准**: 所有测试通过

**推荐工具**: GitHub Copilot (测试代码生成)

**预计时间**: 2 小时

**依赖**: 任务 #6, 任务 #8

---

#### 任务 #10: 性能优化
**描述**:
- 实现图片懒加载
- 添加 ISR (增量静态再生成)
- 优化 Lighthouse 分数
- 配置 CDN 缓存

**完成标准**: Lighthouse 分数 > 90

**推荐工具**: Claude (性能分析)

**预计时间**: 1.5 小时

**依赖**: 任务 #8

---

#### 任务 #11: SEO 优化
**描述**:
- 添加 Meta 标签
- 生成 sitemap.xml
- 配置 robots.txt
- 实现 Open Graph 标签

**完成标准**: 文章能够被搜索引擎正确索引

**推荐工具**: Claude (SEO 配置)

**预计时间**: 1 小时

**依赖**: 任务 #10

---

#### 任务 #12: 部署到 Vercel
**描述**:
- 配置 Vercel 项目
- 设置环境变量
- 配置数据库连接
- 设置自定义域名

**完成标准**: 网站能够通过域名访问

**推荐工具**: Vercel CLI

**预计时间**: 30 分钟

**依赖**: 任务 #11

---

TaskPilot 会自动：

3. 在侧边栏显示 Task Board
