# PR: Proma 四大核心功能增强

## 📋 概述

本 PR 为 Proma 添加了四个核心功能增强，显著提升用户体验和工作效率：

1. **TodoList 侧边栏** - 独立的任务管理面板
2. **批量权限管理** - 一键处理多个权限请求
3. **MCP Server 市场** - 类似 VS Code 扩展市场的 MCP 安装器
4. **代码追踪功能** - 实时追踪 Agent 执行过程

## 🎯 功能详情

### 1. TodoList 侧边栏 ⭐⭐⭐⭐⭐

**问题**：现有的 Task 系统只在 Team 面板中显示，缺少专门的任务管理界面。

**解决方案**：
- 在 SidePanel 中添加独立的 "TodoList" Tab
- 支持短期/长期任务分组
- 支持任务创建、编辑、删除、完成
- 支持拖拽排序
- 显示任务统计和进度

**文件变更**：
- `apps/electron/src/renderer/components/agent/TodoListPanel.tsx` (新增)
- `apps/electron/src/renderer/components/agent/SidePanel.tsx` (修改)
- `apps/electron/src/renderer/atoms/agent-atoms.ts` (修改)

### 2. 批量权限管理 ⭐⭐⭐⭐⭐

**问题**：多个权限请求弹窗需要逐个处理，效率低下。

**解决方案**：
- 批量选择允许/拒绝
- 按类型分组显示（文件操作、网络请求、系统命令等）
- 记住用户选择，自动应用规则
- 支持正则表达式匹配

**实施计划**：
- [ ] 设计批量权限管理 UI
- [ ] 实现权限分组逻辑
- [ ] 实现规则记忆功能
- [ ] 添加正则表达式支持

### 3. MCP Server 市场 ⭐⭐⭐⭐⭐

**问题**：安装 MCP Server 需要手动编辑 JSON 配置，门槛高。

**解决方案**：
- 类似 VS Code 扩展市场的界面
- 浏览、搜索、分类 MCP Servers
- 一键安装和配置
- 管理已安装的 Servers
- CLI 命令支持

**实施计划**：
- [ ] 设计 MCP 市场 UI
- [ ] 实现 Server 列表和搜索
- [ ] 实现一键安装功能
- [ ] 添加 CLI 命令支持

### 4. 代码追踪功能 ⭐⭐⭐⭐

**问题**：Agent 执行过程不透明，难以调试和理解。

**解决方案**：
- 实时显示执行的代码和命令
- 显示执行结果和输出
- 支持断点和单步调试
- 性能分析和历史回放
- 导出日志

**实施计划**：
- [ ] 设计代码追踪 UI
- [ ] 实现实时代码显示
- [ ] 实现执行历史记录
- [ ] 添加调试功能
